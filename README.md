# golang-API-Backend

### GOLANG Todo Web Api with Node project Struct

 Do Git Clone and if you wanna create new go.mod delete the existing .mod file and then 
```
go mod init todo/v1
```
### Create .env file with following fields

```
PORT=8080
DB_HOST=127.0.0.1
DB_USER=root
DB_PASS=root

```
### Run the Program
```
go run main.go
```
